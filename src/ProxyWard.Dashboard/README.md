# ProxyWard Dashboard

React, TypeScript, and Vite operator console for `ProxyWard.Management.Api`.

## Scripts

```bash
npm run dev
npm run typecheck
npm run build
npm run test:e2e
npm run preview
```

Set `VITE_PROXYWARD_API_BASE_URL` to point the dashboard at a management API instance. The default is `http://localhost:8081`.
